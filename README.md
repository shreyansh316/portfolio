Welcome to the source repository for my professional portfolio. Engineered with a strict **"Anti-Gravity"** visual architecture, this platform serves as an interactive demonstration of my capabilities in Systems Infrastructure, Algorithmic Logic, and Intelligent Systems (AI/NLP).

## 🌌 The Aesthetic Engine

This portfolio strictly avoids playful trends and neon gradients in favor of a highly technical, mature, and sophisticated environment:
- **The Void**: A continuous WebGL background rendered via Three.js (`@react-three/fiber`). Abstract geometric primitives float and rotate in deep space, reacting dynamically to cursor lighting vectors.
- **Physics**: UI elements utilize continuous, asynchronous Y-axis floating animations via Framer Motion, creating a weightless aesthetic.
- **Magnetic Fluidity**: A custom dual-element cursor engine repels and snaps to interactive nodes, complemented by hardware-accelerated Lenis smooth scrolling.

## 🛠️ Integrated Nodes

1. **Algorithmic Intelligence**: Live dashboards linking directly to my [LeetCode](https://leetcode.com/u/shreyanshchoolet/) and [CodeChef](https://www.codechef.com/users/goofy_cape_08) profiles, demonstrating rigorous C++ data structures and logic capabilities.
2. **Infrastructure & Cloud Matrix**: An interactive terminal environment detailing my enterprise-level experience with RHEL, Ubuntu Server, and AWS Cloud Management (EC2, S3, VPCs).
3. **High-Fidelity Prototyping**: The *Interactive Sandbox* featuring a stylized, floating CSS/HTML mockup of a Windows 12 Pro Clone.
4. **Intelligent Systems**: Floating magnetic project cards highlighting my work on the **ShikshaSetu AI System** (NLP/Streamlit), **Satellite Pollution Engine**, and the **AlumNET Ecosystem** (PostgreSQL/Auth).
5. **Operations & Logistics**: Demonstrating hands-on leadership, supply chain logic, and tournament coordination.
6. **Data Node Simulation**: An interactive terminal simulating the real-time NLP parsing of unstructured financial transactions into structured data logic.

## 🚀 Deployment Logistics

To spin up this architecture locally:

```bash
# Clone the repository
git clone https://github.com/shreyansh316/portfolio.git

# Navigate to the matrix
cd portfolio

# Install dependencies (NPM)
npm install

# Initialize the development server
npm run dev
```

The application is fully configured for Vercel edge deployment.
